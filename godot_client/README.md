# Echo-Sim Godot Client

## Запуск

### 1. Запусти сервер на ПК
```bash
python echo_sim/server.py
```
Сервер стартует на `http://localhost:8080`

### 2. Открой проект в Godot 4
Открой папку `godot_client` как проект Godot.

### 3. Играй на ПК
В диалоге подключения оставь `http://127.0.0.1:8080` и нажми "Подключить".

---

## Игра с телефона

Нужно сделать сервер доступным извне. Два способа:

### Вариант A — ngrok (проще)
1. Скачай [ngrok](https://ngrok.com/download)
2. Запусти туннель:
   ```bash
   ngrok http 8080
   ```
3. Скопируй URL вида `https://xxxx.ngrok-free.app`
4. В Godot-клиенте на телефоне введи этот URL

### Вариант B — локальная сеть (телефон и ПК в одном WiFi)
1. Узнай IP своего ПК:
   ```bash
   ipconfig  # Windows
   ```
   Ищи строку `IPv4 Address` — например `192.168.1.100`
2. В диалоге подключения введи `http://192.168.1.100:8080`
3. Убедись что брандмауэр Windows не блокирует порт 8080

### Сборка APK для Android
1. В Godot: Project → Export → Add → Android
2. Настрой Android SDK (нужен Android Studio)
3. Export Project → сохрани APK
4. Установи APK на телефон

## Команды в игре
Пиши что делаешь — или используй команды:
- `осмотреться` — описание локации
- `идти <id>` — перейти (tavern, market, alley, castle_gate...)
- `говорить <npc_id>` — поговорить (innkeeper, merchant, guard_captain...)
- `атаковать <npc>` — напасть
- `магазин` — список товаров
- `купить <id>` — купить товар
- `инвентарь` — список вещей
- `статус` — состояние персонажа
- `сохранить` / `загрузить` — сохранение
